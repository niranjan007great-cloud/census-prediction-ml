A machine learning project that analyzes US Census data to predict net migration rates across counties and states.
Built using Python with Pandas, Scikit-learn, Matplotlib, and Seaborn
Dataset includes features like population estimates, births, deaths, and migration rates
Performed label encoding on state and county name columns for model compatibility
Conducted exploratory data analysis (EDA) with scatter plots, histograms, and bar charts
Visualized relationships between census population and estimated base population
Analyzed birth and death trends across 2013–2015 using line and violin plots
Applied PCA (Principal Component Analysis) to reduce dimensionality to 3 components
Used Linear Regression model to predict net migration rate (RNETMIG2015)
Evaluated model using Mean Squared Error (MSE) and Mean Absolute Error (MAE)
Useful for understanding population trends and migration patterns using real US Census data.
